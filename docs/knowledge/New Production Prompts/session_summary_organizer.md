# Session Summary — Organizer

Generate a summary for the organizer based on the full conversation. This summary is shown to the user in the chat. Write it in the user's language AND in English.

---

# Instructions

**Language:** Professional, structured, substantive. Organizers expect and need framework references and actionable analysis.

**Structure the summary as follows:**

1. **Case overview:** Company, location, sector, number of workers affected, timeline, violation type. Concise factual summary.

2. **Framework analysis:**
   - ILO conventions engaged (with numbers and brief description of relevance)
   - OECD provisions if applicable (specify chapters)
   - FSC requirements if applicable
   - CSDDD applicability (company scope, phase)
   - Company's own commitments that create additional leverage

3. **Severity assessment:** Your evaluation of the case — fundamental rights violation, potential violation, or concern outside scope. Brief justification.

4. **Evidence status:** What evidence exists, what gaps remain, what would strengthen the case.

5. **Recommended actions:**
   - Immediate: evidence preservation, worker safety measures if needed
   - Short-term: additional information to gather, follow-up contacts
   - Escalation: recommended pathway (which mechanism, through which channel)
   - All escalation through national union → UNI Global Union

6. **Information gaps:** Specific missing elements that should be collected.

**Tone:** Collegial, direct, actionable.

**Length:** Structured but concise. Prioritize actionable information.

**Do not include:** Session integrity assessment or content that belongs exclusively in the internal UNI document.
